# Data folder (do not commit)

Place any locally extracted analysis tables here, for example:

- `data/mimic_sup_ppi_h2ra.parquet`
- `data/eicu_sup_ppi_h2ra.parquet`

Raw PhysioNet archives (already downloaded in this repo) are kept under:
- `data/raw/physionet/`

Do **not** commit any patient-level data to version control.
